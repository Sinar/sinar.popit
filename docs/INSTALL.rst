sinar.popit Installation
------------------------

To install sinar.popit using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinar.popit`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinar.popit

* Re-run buildout, e.g. with:

    $ ./bin/buildout
